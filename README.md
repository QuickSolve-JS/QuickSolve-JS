# QuickSolve Calculation Engine 🚀

A high-precision, action-based calculation engine for Laravel, specifically designed for POS and financial systems. Built with SOLID principles and BCMath for "Bank-level" accuracy.

## Features
- **Action Pattern:** Modular logic for Tax and Discounts.
- **BCMath Integration:** No more floating-point decimal errors.
- **Fluent Interface:** Easy-to-read method chaining.
- **Unit Tested:** 100% reliable calculations.

# Contributing to QuickSolve-JS

We love contributions! Here is how you can help:

1. **Fork** the repository.
2. **Clone** your fork to your local machine.
3. Create a **Branch** for your feature (`git checkout -b feature/AmazingFeature`).
4. **Commit** your changes (`git commit -m 'Add some AmazingFeature'`).
5. **Push** to the branch (`git push origin feature/AmazingFeature`).
6. Open a **Pull Request**.

### Requirements for PRs:
- All tests must pass (`./vendor/bin/phpunit tests`).
- Follow PSR-12 coding standards.
